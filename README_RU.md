# VoxLocal

**Полностью локальная голосовая диктовка для macOS с приоритетом приватности.** Нажмите глобальное сочетание клавиш, продиктуйте, отпустите — распознанный текст появится в активном приложении. Речь не покидает ваш Mac.

English version: [README.md](README.md)

## Что умеет

- **Приложение в строке меню** (без иконки в Dock) с глобальным сочетанием — по умолчанию **⌥ Option + Пробел**:
  - *удержание*: запись, пока клавиши нажаты; распознавание после отпускания;
  - *переключение*: нажали — запись началась, нажали ещё раз — остановилась.
- **Локальное распознавание** через [whisper.cpp](https://github.com/ggml-org/whisper.cpp) (закреплена версия `v1.9.1`) с ускорением Metal на Apple Silicon. Русский, английский, автоопределение языка.
- **Необязательное улучшение текста** (пунктуация, регистр, слова-паразиты) через **локальный Ollama** на `localhost`. Если Ollama не установлен, недоступен, не уложился в тайм-аут или вернул ерунду — вставляется исходная расшифровка. Диктовка не ломается никогда.
- **Вставка в любое приложение**: сначала через Accessibility API, затем имитация ⌘V с восстановлением буфера обмена, в крайнем случае — «текст скопирован в буфер».
- Компактный плавающий **оверлей**: готов / слушаю (с индикатором уровня микрофона) / распознавание / улучшение / вставка / готово / отменено / ошибка. `Esc` — отмена. Оверлей не забирает фокус.
- **Русский (по умолчанию) и английский** интерфейс.

## Модель приватности

- Звук пишется во временный WAV, распознаётся локально и **сразу удаляется** — после успеха, отмены или ошибки.
- **Без облачных API, ключей и аккаунтов.** Единственный сетевой запрос — загрузка модели Whisper, которую вы запускаете явно, увидев размер.
- Ollama работает только по `http://127.0.0.1:11434` (адрес настраивается, но **разрешён только loopback** — внешние адреса отклоняются).
- **Нет аналитики, телеметрии, трекинга и crash-репортов.**
- В журналах нет аудио, текстов диктовок и содержимого буфера обмена. Подробно: [PRIVACY.md](PRIVACY.md).

## Требования

- macOS **14+** (собрано и проверено на macOS 26, Apple Silicon).
- Рекомендуется Apple Silicon (ускорение Metal). На Intel соберётся без Metal.
- Xcode **Command Line Tools** (или полный Xcode) для сборки.
- Диск: ~500 МБ на артефакты сборки + модель (base ≈ 148 МБ, small ≈ 488 МБ, large-v3 ≈ 3,1 ГБ).
- Опционально: [Ollama](https://ollama.com) с любой instruct-моделью (например, `ollama pull qwen2.5:3b`).

## Сборка и запуск

```bash
git clone https://github.com/romarayt/VoxLocal.git && cd VoxLocal

./scripts/bootstrap.sh    # проверит инструменты, при необходимости скачает проектный cmake,
                          # склонирует whisper.cpp v1.9.1 и соберёт whisper-cli (Metal)
./scripts/test.sh         # автоматические тесты
./scripts/download_model.sh base   # ~148 МБ, спросит подтверждение (можно и из приложения)
./scripts/build_app.sh    # соберёт dist/VoxLocal.app (ad-hoc подпись)
./scripts/run.sh          # запустит приложение
```

Homebrew, CocoaPods и другие глобальные менеджеры пакетов **не требуются** — всё проектно-локально (`vendor/`).

## Первый запуск (разрешения)

Приложение открывает мастер настройки, который проведёт по всем шагам:

1. **Микрофон** — macOS покажет запрос при первой записи; либо Настройки системы → Конфиденциальность и безопасность → Микрофон → включите *VoxLocal*.
2. **Универсальный доступ** (для вставки текста в другие приложения) — Настройки системы → Конфиденциальность и безопасность → Универсальный доступ → добавьте/включите *VoxLocal*. Без него приложение тоже работает: текст копируется в буфер, вставляете сами (⌘V).
3. **Модель** — скачайте `base` (рекомендуется) из приложения (Настройки → Распознавание) или скриптом `./scripts/download_model.sh`.
4. **Ollama (опционально)** — установите с ollama.com, выполните `ollama pull qwen2.5:3b`, включите улучшение в Настройках → Улучшение и выберите модель. Сменить модель можно там же в любой момент.

> **Про ad-hoc подпись:** каждая пересборка меняет подпись, и macOS считает приложение новым — после пересборки может понадобиться заново выдать разрешения Микрофон/Универсальный доступ.

## Настройки

Сочетание и режим (удержание/переключение), микрофон, модель Whisper, язык речи, число потоков, чистка артефактов, улучшение вкл/выкл + адрес + модель + пресет (исходный / чистая диктовка / кратко / деловой / сохранить формулировки / своя инструкция) + тайм-аут, способ вставки (авто / только буфер), автозапуск, язык интерфейса (RU/EN/системный), уровень журнала, сброс онбординга.

## Решение проблем

| Симптом | Решение |
|---|---|
| Сочетание не срабатывает | ⌥Пробел занят другим приложением — VoxLocal покажет предупреждение; выберите другое сочетание в Настройках → Основные. |
| «Нет доступа к микрофону» | Настройки системы → Конфиденциальность и безопасность → Микрофон → включите VoxLocal. |
| Текст не вставился, «скопирован в буфер» | Выдайте разрешение «Универсальный доступ» или вставьте вручную (⌘V). Поля паролей не заполняются автоматически — намеренно. |
| «Модель распознавания не установлена» | Скачайте в Настройках → Распознавание (или `./scripts/download_model.sh base`). |
| «whisper-cli не найден» | Выполните `./scripts/bootstrap.sh`, затем снова `./scripts/build_app.sh`. |
| Улучшение не применяется | Настройки → Улучшение → «Проверить»: сервер Ollama и модель должны существовать. Адрес — только localhost. |
| Медленное распознавание | Возьмите модель поменьше (`base`/`tiny`) или увеличьте число потоков. |
| Сбросить разрешения macOS | `tccutil reset Microphone org.voxlocal.VoxLocal && tccutil reset Accessibility org.voxlocal.VoxLocal` |

Журнал: меню → «Открыть журнал» (`~/Library/Logs/VoxLocal/`, ограниченный размер, без чувствительных данных).

## Известные ограничения

- Ad-hoc подпись ⇒ повторная выдача разрешений после пересборки; на другом Mac Gatekeeper может потребовать «Открыть» через контекстное меню.
- Пока идёт диктовка, `Esc` перехватывается глобально (для отмены); после завершения сессии клавиша сразу освобождается.
- Whisper распознаёт после остановки записи (без потокового частичного результата).
- Редкие приложения, блокирующие синтетическую вставку, получают текст только через буфер обмена.
- `large-v3` на Mac с 8 ГБ может быть медленной; рекомендуются `base`/`small`.

## Удаление

1. Завершите VoxLocal (меню → Завершить).
2. Удалите `dist/VoxLocal.app` (или куда вы его скопировали).
3. Данные (по желанию):
   ```bash
   rm -rf ~/Library/Application\ Support/VoxLocal   # модели
   rm -rf ~/Library/Logs/VoxLocal                   # журналы
   defaults delete org.voxlocal.VoxLocal 2>/dev/null # настройки
   tccutil reset Microphone org.voxlocal.VoxLocal
   tccutil reset Accessibility org.voxlocal.VoxLocal
   ```
